Module actor-abstract-path

Comunica Abstract Path Actor

An abstract Query Operation actor that handles SPARQL property path operations.

This module is part of the Comunica framework, and should only be used by developers that want to build their own query engine.

Click here if you just want to query with Comunica.

Install

 
$ yarn add @comunica/actor-abstract-path

Usage

This package exposes ActorAbstractPath that provides utility methods for actors that implement property path operations.
